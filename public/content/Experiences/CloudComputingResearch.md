### Cloud Computing Researcher | NJIT
**Newark, New Jersey • January 2024 – May 2024**

---

#### **Research & Architecture Overview**
Collaborated with Associate Professor Xiaoning Ding to design and build a fully scalable, cloud-native serverless application on AWS. This project served as a proof of concept for elastic web architecture, focusing on building a system capable of maintaining performance and cohesion under variable global loads.

#### **Key Contributions & Impact**
* **Cloud-Native System Architecture**
    * Architected a full-stack system using **Python** and a suite of AWS services, including **Lambda, API Gateway, S3, EC2, IAM, and DynamoDB**.
    * Engineered the frontend and backend to function as a unified, resilient microservice architecture.
* **Elasticity Validation & Stress Testing**
    * Conducted extensive load testing by distributed execution across five nodes, sending over **50,000 requests**.
    * Achieved sustained rates exceeding **250 requests per second**, with fewer than 5,000 throttles recorded, proving the elasticity of the design.
* **Microservice & API Development**
    * Deployed and invoked over **five REST APIs** using Python and JavaScript to enable efficient, independent service communication.
    * Optimized the infrastructure for cost-effectiveness and production-ready reliability within a serverless environment.
